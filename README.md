# Match Card Game
Minigame on web browser develop by using ReactJS framwork

## Getting Started
#### Prerequisite
 * [**NodeJs version ^8**](https://nodejs.org/en/download/)
 * [**npm**](https://www.npmjs.com/get-npm)
 
 
 #### Runing project
 ##### Dev environment
```
cd app/ // Go to app directory
npm install
npm run dev
```
##### Production environment
```
cd app/ // Go to app directory
npm run production
```
